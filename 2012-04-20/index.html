Moviepilot.com A/B Testing

We changed the structure of A/B testing a bit so we can do more flexible testing.

An anonymous user is now assigned randomly to several A/B testing groups:

  • groupHalf: 50% / 50%
  • groupThird: 33% / 33% / 33%
  • groupQuarter: 25% / 25% / 25% / 25%

–> The fraction numbers start at 0.

–> groupHalf == 0 means the user is in the 1st 50/50 group.

–> groupQuarter == 3 means the user is in the 4th 25/25/25/25 group.
